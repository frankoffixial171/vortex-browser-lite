# 🌀 Vortex Browser

A fast, private Android browser with built-in protection.

## Features

- 🛡️ **Ad Blocker** — Blocks ads from 40+ ad networks
- 🕵️ **Tracker Blocker** — Blocks 35+ tracking/analytics domains
- 🔒 **Fingerprint Protection** — Randomizes canvas, WebGL, and audio fingerprints
- 🔐 **Force HTTPS** — Automatically upgrades all connections to HTTPS
- 🧹 **Clear Data on Exit** — Wipes history, cache, cookies when you close the app
- 🌐 **Proxy Support** — HTTP and SOCKS5 proxy configuration
- 📱 **Bottom Address Bar** — Easy one-handed browsing
- 🏠 **Custom Homepage** — Beautiful Vortex Start page with quick shortcuts
- 🔍 **Multiple Search Engines** — Google, DuckDuckGo, Bing, Brave, Yahoo
- ⬇️ **Download Manager** — Built-in file download support

## Build

```bash
./gradlew assembleDebug
```

APK will be at `app/build/outputs/apk/debug/app-debug.apk`

## Install

Download the latest APK from [Releases](../../releases) and install on your Android device.
